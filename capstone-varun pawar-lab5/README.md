# Travel Explorer - Capstone Project

This is a simple, responsive Travel Agency website made using HTML and CSS.

## Sections Included
- Header & Navigation
- Hero Section with Background Image
- Travel Packages (Grid Layout)
- Reviews Section
- Contact Form
- Footer

## Tools Used
- HTML5
- CSS3 (Grid + Responsive Design)

## How to View
Just open **index.html** in any browser.
